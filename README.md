# Clean Drive GTM v3 Balanced

Static landing page for a WhatsApp-first doorstep car care service in Karachi.

## Files

- `index.html` — main website
- `hero-car-cleaner.webp` — hero image
- `README.md` — deployment notes

## Deployment

Upload the three files to GitHub at the repository root and deploy on Vercel using Framework Preset `Other`. No build command is required.
